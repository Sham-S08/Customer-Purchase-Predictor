# CUSTOMER-PURCHASE-PREDICTOR

**Predict Customer Needs, Boost Sales Instantly**

---

## Overview
This is a Streamlit web app that predicts whether an online shopper is likely to make a purchase using browsing behavior data and a trained machine learning model.
